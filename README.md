# 译间 - 散文翻译练习平台

一个专业的散文翻译练习平台，提供 AI 智能批改、学情分析等功能。

## 功能特性

### 📖 核心功能
- **多篇散文**：包含多篇经典散文，支持逐段翻译练习
- **AI 批改**：智能批改翻译，提供详细反馈
- **错题本**：自动记录翻译错误，支持反复练习
- **术语库**：内置专业术语，支持自定义添加
- **学情分析**：可视化数据展示，追踪学习进度

### 🎯 Demo 体验
- 首页顶部提供 Demo 体验卡片
- 引导式使用流程
- 完整功能预览

### 📊 数据分析
- **总览**：练习统计、学习进度
- **得分分析**：各篇得分对比、段落得分趋势
- **易错分析**：错误类型分布、高频易错词汇
- **进步趋势**：多次练习进步曲线、成就勋章

## 项目文件

```
.
```

## 快速开始

### 直接使用
1. 直接用浏览器打开 `index.html` 即可使用
2. 所有数据保存在浏览器 localStorage 中
3. 支持离线使用

### 上传到 GitHub
1. 在 GitHub 创建新仓库
2. 上传 `yijian-prose-translation.zip` 中的文件
3. 或者直接上传解压后的文件：
   - `index.html`
   - `corpus.json`
   - `README.md`

## 技术栈

- **前端**：原生 HTML/CSS/JavaScript
- **图表库**：Chart.js (CDN)
- **AI 接口**：DeepSeek API (可选)
- **数据存储**：LocalStorage
- **字体**：Google Fonts (Lora, Noto Serif SC)

## 功能说明

### 练习流程
1. 从篇目选择页选择一篇散文
2. 逐段进行翻译练习
3. 提交后查看 AI 批改结果
4. 查看参考译文和注释
5. 继续下一段练习

### 数据分析
- 每次练习自动记录数据
- 学情分析提供可视化图表
- 帮助识别薄弱环节

### 错题复习
- 错题自动添加到错题本
- 支持随机复习模式
- 掌握率统计
