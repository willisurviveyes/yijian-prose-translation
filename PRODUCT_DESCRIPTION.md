# 译间 - English-Chinese Prose Translation Platform

## Product Description

**Yijian (译间)** is a sophisticated online platform designed for English-Chinese translation practice, focusing on classic Chinese prose. Built with a clean, elegant interface and powered by AI technology, it provides comprehensive translation learning and practice experience.

## Key Features

### 📖 Authentic Prose Collection
- Curated selection of classic Chinese prose from renowned authors
- Each piece includes professional English translations (Zhang Peiji style)
- Detailed translator notes and explanations
- Paragraph-by-paragraph learning format

### 🤖 AI-Powered Grading System
- Real-time AI translation evaluation and feedback
- Detailed scoring and grading
- Error identification and correction suggestions
- Comprehensive grammar, spelling, and vocabulary checks
- Professional translation style guidance

### 📚 Smart Learning Tools
- **Error Notebook**: Automatically collects translation mistakes for review
- **Terminology Library**: Built-in professional terms + custom additions
- **Analytics Dashboard**: Visual progress tracking and performance analysis
- **Study Notes**: Keep track of important learnings

### 📊 Comprehensive Analytics
- **Overview Dashboard**: Practice statistics, average scores, completion rates
- **Score Analysis**: Comparative charts for each prose piece
- **Error Analysis**: Visual breakdown of common mistakes
- **Progress Tracking**: Improvement curves and achievement system

## User Experience

### Modern, Clean Design
- Forest green theme for comfortable reading
- Elegant typography with Noto Serif SC and Lora fonts
- Responsive interface for all devices
- Smooth animations and intuitive navigation

### Guided Learning Journey
- Interactive demo for first-time users
- Step-by-step practice workflow
- Personalized learning recommendations
- Achievement system to motivate progress

### Data-Driven Learning
- All practice data saved locally in browser
- Visual charts powered by Chart.js
- Weakness identification and improvement suggestions
- Long-term progress monitoring

## Technology Stack
- **Frontend**: Pure HTML/CSS/JavaScript
- **Visualization**: Chart.js
- **AI Integration**: DeepSeek API
- **Storage**: Browser LocalStorage
- **Fonts**: Google Fonts (Lora, Noto Serif SC)

## Who Is It For?

- Translation students and professionals
- Chinese prose enthusiasts
- Anyone preparing for translation exams
